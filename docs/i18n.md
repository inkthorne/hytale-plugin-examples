# Localization (i18n) API

Events for the internationalization/localization system.

---

## GenerateDefaultLanguageEvent

**Package:** `com.hypixel.hytale.server.core.modules.i18n.event`

Implements `IEvent<Void>`. Fired when generating default language translations.

| Method | Return Type | Description |
|--------|-------------|-------------|
| `putTranslationFile(String, TranslationMap)` | `void` | Add translation file |

---

## Usage Example

```java
import com.hypixel.hytale.server.core.modules.i18n.event.GenerateDefaultLanguageEvent;

@Override
protected void setup() {
    // Register default translations
    getEventRegistry().register(GenerateDefaultLanguageEvent.class, event -> {
        TranslationMap translations = new TranslationMap();
        translations.put("my_plugin.greeting", "Hello, world!");
        translations.put("my_plugin.farewell", "Goodbye!");

        event.putTranslationFile("my_plugin", translations);
    });
}
```

---

## Notes

- Use this event to register plugin translations during server startup
- Translation keys should be namespaced with your plugin name to avoid conflicts
- Translations are used for player-facing messages and UI elements
